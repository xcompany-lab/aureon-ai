## TL;DR [AH001_006, AH001_007, AH001_009]

A estrutura correta de um time de vendas outbound é a **Christmas Tree**: BDRs na base (volume), SDS no meio (qualificação + valor), BCs no topo (fechamento). Especialização de papéis é o que transforma volume em receita previsível.

---

## Consenso entre Fontes [AH001_006, AH001_009, AH001_012]

**Especialização aumenta performance coletiva.** Um closer tentando prospectar, qualificar e fechar ao mesmo tempo faz cada uma dessas atividades pior do que um especialista. A separação de papéis não é overhead — é multiplicador.

---

## A Christmas Tree Structure [AH001_006, AH001_007, AH001_008, AH001_009, AH001_010]

```
              ┌──────────────────────┐
              │         BC           │
              │  Business Consultant │  → Fecha com CLOSE Framework
              │  Base baixa +        │
              │  comissão agressiva  │
              └──────────┬───────────┘
                         │ (1 BC : 2 SDS)
         ┌───────────────┴───────────────┐
         │             SDS               │
         │  Sales Dev Specialist         │  → Call valor 30min
         │  Base + bônus por transição   │     Qualifica + entrega
         │  aceita                       │     Transição scripted
         └───────────────┬───────────────┘
                         │ (1 SDS : 2 BDRs)
   ┌─────────────────────┴─────────────────────┐
   │                   BDR                     │
   │  Business Development Rep                 │  → 100 calls/dia
   │  Base + bônus por agendamento qualificado │     30% contact rate
   │                                           │     10% set rate
   └───────────────────────────────────────────┘
```

**Proporção alvo:** 2 BDRs → 1 SDS → 0.5 BCs

## Perfis por Cargo [AH001_006, AH001_009, AH001_012, AH001_020]

| Cargo | Perfil Ideal | Métrica Principal | Modelo de Comp |
|-------|-------------|-------------------|----------------|
| BDR | Alta energia, baixo ego, resiliente | 3 agendamentos/dia | Base + bônus appointment |
| SDS | Empático, curioso, bom ouvinte | Show rate 60%+ | Base + bônus transição |
| BC | Confiante, dominante, high conviction | Close rate | Base baixa + comissão alta |

## O BDR como Mídia Mensurável [AH001_006]

A resistência ao custo do BDR some quando você reframe o papel: **BDR é propaganda**.

```
CUSTO BDR MENSAL = $X
AGENDAMENTOS/MÊS = Y
CPL (Custo por Lead) = $X/Y

Compare com Google Ads CPL.
Se BDR for mais barato → ESCALAR BDRs.
Se Ads for mais barato → ESCALAR ADS.
```

Ambos são canais de aquisição. O BDR tem a vantagem de ser um ser humano que aprende, itera e se adapta.
