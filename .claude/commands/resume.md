# RESUME SESSION

Carrega o contexto da última sessão salva.

## Instruções

1. Leia `/logs/LIVE-SESSION/CURRENT-SESSION.md`
2. Leia `/logs/LIVE-SESSION/CONTEXT.json`
3. Apresente ao usuário:

```
[RESUMING SESSION]

## Última Atividade
{resumo do que foi feito}

## Status
{status da tarefa: in_progress/completed/blocked}

## Pendências
{lista de pendências}

## Próximos Passos
{próximos passos sugeridos}

## Arquivos Relevantes
{lista de arquivos importantes}

---
Quer continuar de onde parou?
```

4. Aguarde confirmação do usuário antes de continuar.
