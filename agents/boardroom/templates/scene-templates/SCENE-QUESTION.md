# ❓ CENA: PERGUNTA AO OUVINTE
## Template para Fechamento com Provocação

---

## PROPÓSITO

Todo episódio termina com uma **pergunta direta ao ouvinte**. Isso transforma o episódio de entretenimento passivo em **gatilho de ação**.

---

## ESTRUTURA

```
[NARRADOR] → Resume aprendizados (3 pontos)
[NARRADOR] → Lista fontes
[PAUSA]
[NARRADOR] → Pergunta direta ao ouvinte
[NARRADOR] → Desdobramento ou segunda pergunta
[ENCERRAMENTO]
```

---

## TIPOS DE PERGUNTAS

### Pergunta de Ação
```
"E você?

O que você vai fazer amanhã de manhã
sobre a estrutura de comissão do seu time?

Não semana que vem. Amanhã."
```

### Pergunta de Reflexão
```
"E você?

Está pagando bem pessoas medianas
ou pagando mal pessoas excelentes?

Qual dos dois erros está cometendo?"
```

### Pergunta de Diagnóstico
```
"E você?

Se seu melhor vendedor pedisse demissão hoje,
você saberia dizer exatamente por quê?

Ou só descobriria na entrevista de desligamento?"
```

### Pergunta de Confronto
```
"E você?

Está resolvendo o problema real
ou o problema que é mais fácil de resolver?

Seja honesto."
```

### Pergunta de Próximo Passo
```
"E você?

Dos três modelos discutidos hoje,
qual mais se aproxima da sua realidade?

E o que te impede de implementar na segunda-feira?"
```

## EXEMPLO COMPLETO

```
[NARRADOR]
(tom reflexivo)
"A reunião acabou. Mas o aprendizado continua.

O que você observou:

1. O problema apresentado raramente é o problema real.
   CRO queria comissão. O problema era qualidade do time.

2. Contexto muda tudo.
   O que funciona nos EUA precisa de adaptação para o Brasil.

3. Decisão boa é decisão que sobrevive ao Devil's Advocate.
   Se não aguenta questionamento, não aguenta a realidade.

As fontes desta decisão:
- Cole Gordon: Estrutura híbrida de compensação
- Alex Hormozi: Limpeza de time antes de aumentar comissão
- DNA Cognitivo: Princípio de alinhamento de incentivos"

[PAUSA 2 seg]

[NARRADOR]
(tom direto)
"E você?

Quantos vendedores medíocres você está pagando
pra ficarem felizes sendo medíocres?

E mais importante: se você limpasse o time amanhã,
teria coragem de aumentar a comissão dos que ficassem?

Essa é a pergunta que fica."

[SOM: Porta abrindo]

[NARRADOR]
"Até o próximo episódio de Boardroom Warfare."

[SOM: Passos se afastando. Música de fechamento - 5 seg.]
```

---

## REGRAS

1. **Sempre pessoal** - "E você?", não "E as empresas?"
2. **Sempre específica** - Relacionada ao tema do episódio
3. **Sempre provocativa** - Deve incomodar um pouco
4. **Máximo 2 perguntas** - Mais que isso dilui o impacto
5. **Terminar com "Essa é a pergunta que fica"** - Âncora de fechamento
