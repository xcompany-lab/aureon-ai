# ⚔️ CENA: DEBATE DOS AGENTS
## Template para Confrontos entre Agents of Person

---

## PROPÓSITO

O coração do episódio. Aqui os **Agents of Person** (digital twins dos experts) debatem com suas personalidades, gírias e perspectivas únicas.

---

## DINÂMICAS DE DEBATE

### Padrão de Confronto
```
[AGENT-A] → Afirma posição
[AGENT-B] → Contesta ou complementa
[CITADOR] → Referencia fonte
[AGENT-A] → Defende ou ajusta
[NARRADOR] → Explica o que aconteceu (breve)
```

### Padrão de Interrupção
```
[AGENT-A] → Começa a falar
[AGENT-B] → (interrompendo) Corta e contesta
[AGENT-A] → Reage à interrupção
```

### Padrão de Aliança
```
[AGENT-A] → Faz afirmação
[AGENT-B] → Concorda e amplifica
[AGENT-C] → Contesta os dois
```

## REGRAS DO DEBATE

1. **Cada agent só fala o que seu DNA permite**
2. **Gírias características devem aparecer**
3. **Interrupções são naturais** (especialmente HORMOZI)
4. **Citador entra após claims factuais**
5. **Narrador explica dinâmicas** (não opina)
6. **Tensão é produtiva** - não resolver fácil demais
7. **Mínimo 3 rodadas** antes de qualquer convergência
