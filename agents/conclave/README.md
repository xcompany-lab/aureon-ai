# Conclave (L1)

Constitutional deliberation agents for multi-perspective analysis.

## Agents

| ID | Role | Perspective |
|----|------|-------------|
| critico-metodologico | Analytical Guardian | Rigorous analysis, detail-oriented |
| advogado-do-diabo | Devil's Advocate | Challenge assumptions, contrarian |
| sintetizador | Integration Architect | Unify perspectives, find consensus |

## Activation

Use `/conclave` to start a council session with all three agents.

## Protocol

1. Each agent presents their perspective
2. Critic analyzes methodology and gaps
3. Devil's Advocate challenges assumptions
4. Synthesizer integrates into unified recommendation
